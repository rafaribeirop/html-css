Formatação de texto

Negrito

Destacar este termo - tag <b>(forma não semântica)

Destacar este termo - tag <strong> (forma semantica)

Itálico

Destacar este termo - tag <i> (forma não semântica)

Destacar este termo - tag <em> (forma semântica)

Texto marcado

Para marcar um texto usando a tag MARK

(Da pra mudar a cor de cada um individualmente, ou todos usando as CSS.)
Com a CSS todos ficarão da cor desejada com mais facilidade

Texto grande e pequeno

Para texto grande a tag seria BIG, porém ela estpá obsoleta, já que é uma forma NÃO SEMÂNTICA

Entretanto a tag SMALL ainda pode ser usada, já que possui função semântica

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido mas nÃo considerado - tag DEL

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi incluido depois - tag INS


Tambem existe a forma não semântica - U

Texto sobrescrito

Para escerver coisas do tipo X20+3 - tag SUP

Texto subscrito

Para escrever coisas do tipo H2O tag - SUB
